Дисциплина «Русский язык и культура речи»

Билеты для зачета, 1 курс

Исторический факультет, дневное и вечернее отделения, Ст. преп., к.ф.н. Данилкова Ю.Ю.

1Билет №1

  1. Понятие «современный русский литературный язык».
  2. Поставьте ударение: бело, ясно, легки, правы, красивее.
  3. Объясните значение слов с разными ударениями: характерный – характерный.
  4. Замените, где нужно, иноязычные слова русскими: В сочинении ученика много дефектов.
  5. Школьная футбольная команда потерпела фиаско. На концерте превалировали вокальные номера. Подберите определения, правильно согласуйте их с приведенными ниже словами: рояль, кофе.
  6. Выберите нужную форму: Два лучши… друга.

2Билет №2

  1. Орфоэпическая норма русского языка.
  2. Поставьте ударения: газопровод, договор, дремота, кремень, ломоть.
  3. Объясните значение слов с разными ударениями: склонен – склонён.
  4. Замените, где нужно, иноязычные слова русскими: После долгих дебатов договорились назначить прогулку на ближайшее воскресенье. Во время интервала между двумя уроками в класс вошел директор школы. Хозяин решил презентовать своему гостю старые часы.
  5. От предложенных наименований национальностей в м.р. образуйте соответствующие наименования ж.р.: араб, грек, индеец.
  6. Выберите нужную форму: Две новы… модели автомобиля.

3Билет №3

  1. Вариативные нормы русского языка.
  2. Поставьте ударения: мусоропровод, ходатайство, апостроф, иконопись, похороны.
  3. Объясните значения слов с разными ударениями: бронировать – бронировать.
  4. Определите значения следующих слов, составьте словосочетания: амплуа, апогей, апофеоз.
  5. От предложенных наименований национальностей в м.р. образуйте соответствующие наименования ж.р.: индиец, китаец, кореец. Выберите нужную форму: Три наши… спортсмена.

4Билет №4

  1. Лексическая норма русского языка.
  2. Поставьте ударения: упрочение, феномен, бензопровод, ветеринария, генезис, догмат.
  3. Объясните значения слов с разными ударениями: броня – броня.
  4. Определите значения следующих слов, составьте словосочетания: дилемма, дилетант, заурядный.
  5. Поставьте имена существительные в родительном падеже множественного числа: апельсин, ботинок, валенок.
  6. Выберите нужную форму: Четыре больши… квартиры.

5Билет №5

  1. Морфологическая норма: имя существительное.
  2. Поставьте ударения: жизнеобеспечение, каталог, квартал, некролог, нефтепровод.
  3. Объясните значения слов с разными ударениями: языковый – языковой.
  4. Определите значения следующих слов, составьте словосочетания: клише, когорта, компетентный.
  5. Поставьте имена существительные в родительном падеже множественного числа: дупло, кочерга, мандарин.
  6. Выберите нужную форму: Большинство спортсменов (отсутствовало, -ли).

6Билет №6

  1. Морфологическая норма: имя числительное.
  2. Поставьте ударения: искра, кладовая, премирование, пурпур, путепровод.
  3. Объясните значения слов с разными ударениями: угольный – угольный.
  4. Определите значения следующих слов, составьте словосочетания: лаконичный, миссия, нивелировать.
  5. Поставьте имена существительные в родительном падеже множественного числа: басня, вафля, свадьба.
  6. Выберите нужную форму: Часть студентов (пришло, -ли) на экзамен.

7Билет №7

  1. Морфологическая норма: имя прилагательное.
  2. Поставьте ударения: ракурс, столяр, узаконение, христианин, безудержный. Объясните значение слов с разными ударениями: погруженный – погружённый.
  3. Определите значения следующих слов, составьте словосочетания: огульный, оптимальный, ореол.
  4. Поставьте имена существительные в родительном падеже множественного числа: помидор, гектар, баклажан.
  5. Выберите нужную форму: Тридцать студентов (стояло, -ли) в стороне.

8Билет №8

  1. Синтаксическая норма русского языка.
  2. Поставьте ударения: единовременный, зубчатый, лубочный, одновременный гербовый.
  3. Объясните значения слов с разными ударениями: видение – видение.
  4. Определите значение следующих слов, составьте словосочетания: педант, парадокс, палитра.
  5. Поставьте в нужную форму: (среди) татары, армяне, грузины.
  6. Выберите нужную форму: Тридцать один человек (пришел, -ли) на выставку.

9Билет №9

  1. Орфоэпическая норма русского языка.
  2. Поставьте ударения: искристый, мизерный, украинский, баловать, закупорить.
  3. Объясните значения слов с разными ударениями: кирка – кирка.
  4. Определите значения следующих слов, составьте словосочетания: прозорливый, преемник, приоритет.
  5. Поставьте в нужную форму: (среди) калмыки, цыгане, башкиры.
  6. Поставьте нужную форму: Три новы… рубашки.

10Билет №10

  1. Лексическая норма русского языка.
  2. Поставьте ударения: запломбировать, втридорога, втридешева, донельзя, мастерски.
  3. Объясните значения слов с разными ударениями: языковый – языковой.
  4. Определения значения следующих слов составьте словосочетания: фиктивный, форсировать, эквивалент.
  5. Поставьте в нужную форму: (среди) турки, киргизы, буряты.
  6. Поставьте нужную форму: Три соседни… здания.
