Описание проекта

Данный проект является глубокой переработкой сайта РУсланка.РУ (статьи/билеты/заметки на историческую тематику):

  • дизайн сайта адаптирован для iPad.
  • навигация по сайту полностью переделана. Посредством верхнего меню осуществляется переход на страницу-каталог страниц данного раздела, а меню слева позволяет выбрать страницу посредством постепенного уточнения параметров (например, Год и Месяц для Заметок).
  • добавлен поиск по сайту.
  • теперь понравившуюся страницу сайта можно публиковать в социальных сетях (Одноклассники, ВКонтакте, Мой Мир, Я.ру, Живой Журнал, Facebook и так далее).
  • все ресурсы сайта опубликованы под лицензией CC-BY-SA, то есть ресурсы сайта можно использовать на свое усмотрение при соблюдении двух условий: указание авторства и распространение под этой же лицензией. Соотвествующую картинку-ссылку можно увидеть внизу левого меню.
  • все страницы сайта статические, что существенно расширяет список бесплатных хостингов (на данный момент это, github.com).
  • экспорт заметок из ЖЖ и Wordpress по RSS позволяет динамически добавлять новости на статический сайт.
  • экспорт ссылок пока в разработке, но планируется использовать Memori.
