Изучаем условные операторы в Ruby. Часть 1
Главная Заметки История Билеты Глоссарий Оружие Литература Авторы Ссылки

Каждый раз, когда мы применяем условные операторы, мы спрашиваем: это истинно или ложно? И когда получаем ответ — true или false — выполняется соответствующий блок кода.

Что я узнала об операторе if…

Начнем с примера:

if 2 == 2 then
  print "True!"
end

Если правда, что 2 равно (==) 2, а оно так и есть, то оператор if возвратит true, что приведет к выполнению блока кода, состоящего из одного оператора print.

Теперь создадим переменную и сравним ее с числом.

x = 144
if x == 144
  puts "x equals 144"
end

Здесь мы выбросили ключевое слово then из оператора if. В этом примере нет нужды его использовать.

Также можно изменить порядок вещей следующим образом:

x = 144
puts "x equals 144" if x == 144

Когда мы вот так меняем порядок, if становится модификатором оператора.

Есть еще один способ компоновки оператора if, он состоит в замещении then двоеточием (:) :

x = 144
if x == 144: puts "x equals 144" end

Итак, вперед, тренироваться! Стоит поиграться с кодом, чтобы понять «что к чему».

Теперь протестируем истинность или ложность с помощью других операторов. Например, оператор &&, который означает «и»:

ruby = "trendy"
programming = "gaily"

if ruby == "trendy" && programming = "gaily"
  puts "Keep prigramming!"
end

Получим при выводе: Keep programming!. Код в блоке выполняется, если истинные оба выражения.

С помощью && можно проверить несколько выражений;

if a == 11 && b == 28 && c == 44 && d == -15
  print sum = a + b + c + d
end

Сумма будет напечатана, если все эти выражения истинны. Также вместо && можно использовать ключевое слово and.

Есть еще один оператор || (синоним для этого оператора — or). Когда используется || или or, код выполняется, если хотя бы одно из выражений возвращает true:

if ruby == "trendy" or programming == "gaily"
  puts "Keep programming!"
end

Итак, || и or, && и and называются логическими операторами. В Ruby есть множество других операторов, например:

!= — не равно
> — больше чем
< — меньше чем
>= — больше или равно (не меньше)
<= — меньше или равно (не больше)
! и not — меняют значение проверки
